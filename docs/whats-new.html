SpendingTracker
v1.4
April 2026
🔧 Audit Remediation — v1.4

JSON backup restore path added

The Backup modal now has a full restore tab. Load a JSON backup file via file picker or paste — your transactions, accounts, and categories are restored in one step. Previously the module was export-only with no way to recover data on a new device.

Profile schema fix — avatarColor

Profiles created in SpendingTracker now correctly store the avatar color as avatarColor, matching all other modules. Previously, profiles created here would show the default indigo avatar in DebtTracker, IncomeTracker, and other modules.

Responsive layout

SpendingTracker now adapts to mobile, tablet, and desktop screen sizes using the same responsive system as DebtTracker. Nav labels hide on small screens, stat rows reflow, and content columns stack on mobile.

API key show/hide + status indicator

The 🔑 API key field now has a show/hide toggle so you can verify your key before saving. The nav bar shows a colored dot indicating whether your key is valid, invalid, or rate-limited — no more guessing why AI isn't responding.

Recovery PIN in profile setup

The first-run profile creation screen now includes a Recovery PIN field, matching all other modules. Set a memorable word or phrase to recover your data on any device.

⚡ Features — v1.3

Multi-month CSV import

Queue multiple statement files for the same account and import them in one pass. Each file goes through the full flow — column mapping, sign normalization, dedup, and AI categorization — with a progress indicator showing status per file.

Dedup preview before import

Before committing an import, a preview screen shows every transaction labeled New (green), Duplicate (grey), or Conflicting (amber — same date and description but different amount). Skip-all-duplicates is on by default. Conflicting transactions always require an explicit choice: keep existing, overwrite, or import both.

Batch delete

Toggle Select mode in the Transactions tab to get checkboxes on every row. Select All respects your active filters — only visible transactions get selected. A sticky action bar shows the count and a Delete Selected button. Confirmation modal required, no accidental deletes.

Date range picker

Toggle between Month view (existing) and Range view. Range mode adds start and end month pickers — both the Transactions and Summary tabs update to reflect the selected range. The header label shows the full range (e.g. "Jan 2026 — Mar 2026") so you always know what you're looking at.

Trends tab — spending over time

A new fourth tab shows your spending history as SVG charts. A line chart shows total expenses by month. A grouped bar chart breaks down the top 6 categories month by month with an "Other" bucket for the rest. The window is configurable from 3 to 12 months. Click any month bar to jump to that month's transactions.

3-month rolling summary panel

A new panel at the top of the Summary tab shows the last three months side by side — total income, total expenses, and net for each. Delta arrows between columns show whether things are improving or getting worse. Click any column to switch to that month.

⚡ Features — v1.2

Summary tab is now first

Tab order is now Summary / Transactions / Rules with Summary as the default. Your spending overview is the first thing you see when you open the module.

Category bar charts with budget average overlay

Each category now shows a horizontal bar sized as a percentage of your monthly income. Income categories appear at the top, expenses sorted highest to lowest. A dash marks your 3-month rolling average — your visual budget gauge. Delta labels show green, amber, or red depending on how far you're over or under.

Click category → jump to filtered transactions

Clicking any category row in the Summary tab switches to Transactions and pre-filters to that category automatically. No manual filter setup needed.

Debit / Credit toggle + multi-category filter

The Transactions tab now has an All / Debit / Credit toggle and a multi-category selector. Pick one or more categories at once — both filters work together with AND logic. Selected category count shows as a badge.

Create rule or new category from Edit Transaction

"+ Create Rule from This" pre-fills a new rule with the transaction description and applies it retroactively on save. "+ New Category" opens an inline form to create a custom category — name, icon, type, color — and immediately assigns it to the transaction.

Transfer category added

A Transfer category (↔️) is now in the default category set for tracking account-to-account transfers. Automatically seeded into existing profiles on next load.

Dark mode contrast fixes

Account nicknames, Rules tab headers, and empty-state text were using near-invisible grey in dark mode. All upgraded to the correct text color tokens.

🔧 Bug Fixes — v1.1

Profile switcher now works

The nav bar avatar button was not opening the profile dropdown. A new ProfileDropdown component now shows all profiles with an active checkmark. Switching profiles reloads all data — accounts, transactions, categories, and rules — for the selected profile.

JSON and CSV export now working

Both export buttons were silently failing. The download anchor was detached from the DOM before it could fire. Fixed by appending the anchor to the document body before clicking and removing it after. Downloads now work reliably across all browsers.

Flagged transactions now clearly surfaced

After import, low-confidence AI assignments now show an amber left border and a REVIEW badge on each flagged transaction row. A dismissible amber banner in the Transactions tab shows the total count of unreviewed items so nothing slips through.

Rules now apply retroactively

Creating or editing a rule now immediately re-runs against all existing unlocked transactions. A confirmation shows how many transactions were updated. Confirming a Needs Review item also auto-creates a rule and applies it retroactively — next month's import will be faster automatically.

✨ New Module — v1.0

CSV import from any bank or credit card statement

Import spending data from Chase, Bank of America, Amex, or any institution. The column mapper lets you match your CSV's columns to the required fields — and remembers your setup for every future import from that account.

Sign-flip toggle per account

Credit card statements show purchases as positive — bank statements show them as debits. A per-account flip toggle normalizes everything automatically and is remembered for every future import.

AI batch categorization on import

After rules run, all uncategorized transactions are sent to AI in a single API call. High-confidence matches are assigned silently. Low-confidence ones go to a Needs Review queue — confirm them and rules are auto-created for next time.

Rules engine — full CRUD + AI suggest

SpendingTracker owns the shared platform rules engine. Create rules manually, create them from any transaction row, or let AI suggest rules for your unruled merchants. Rules run before AI on every future import.

Actuals view with 3-month rolling average

The Summary tab shows what you actually spent per category this month alongside a 3-month rolling average. Delta is color-coded — green means under average, amber is 10–25% over, red is 25%+ over. No manual budget entry required.

Three-tab layout: Transactions / Summary / Rules

Transactions shows your full monthly list with search and category filter. Summary shows category-level actuals vs rolling average. Rules is the full rules management interface.

Deduplication on import

Before importing, the module checks for existing transactions with matching date, description, and amount. Duplicates are flagged so you can choose to skip or overwrite — safe to re-import the same statement without doubling entries.

Category seeding fallback

If IncomeTracker hasn't been run yet, SpendingTracker seeds the shared FFP category list (56 categories, 13 sections) automatically on first run. The category system is always ready regardless of module order.

IncomeTracker

🔧 Audit Remediation — v1.1

Responsive layout

IncomeTracker now adapts to mobile, tablet, and desktop using the same responsive system as DebtTracker. Nav labels hide on small screens and content reflows correctly on any device.

File upload for JSON import

The backup import now accepts a file picker in addition to paste. Tap "Load from file," select your JSON backup, and it populates automatically — no more copying large JSON strings on mobile.

CSV export

Export your income streams as a CSV file — one row per stream with name, type, frequency, amount, monthly equivalent, stability rating, and dates. Open directly in Excel or Google Sheets.

API key show/hide + status indicator

The 🔑 key field now has a show/hide toggle and the nav bar shows a live status dot for your API key — valid, invalid, or rate-limited.

✨ New Module — v1.0

Track every income stream in one place

Add any income source — W2, self-employment, rental, dividends, side business, benefits, or other. Each stream stores amount, frequency, stability rating, after-tax flag, start/end dates, category, and color.

Automatic monthly normalization

Enter your actual pay amount at whatever frequency you're paid — weekly, bi-weekly, semi-monthly, monthly, quarterly, annual, or one-time. The dashboard normalizes everything to a monthly total automatically.

Stability ratings

Mark each stream as Stable, Mostly Stable, Variable, or Irregular. The dashboard shows your overall income stability at a glance — and this data will feed directly into the AI Advisor when it launches.

Summary dashboard

Total monthly income, annual total, breakdown by income type, and stream count — all calculated live. One-time income is shown separately so it doesn't inflate your recurring total.

Seeds the shared category system

On first run, IncomeTracker seeds the platform-wide category list (56 categories across 13 sections) used by all FFP modules. Every module that launches after this — Spending, AI Advisor — reads from the same shared list.

Shared profile + API key system

Uses the same cc_profiles and cc_apikey storage as all other modules. Set up once and it works across the entire platform.

Export JSON + CSV, Import JSON

Full JSON backup and restore (replace or merge). CSV export for spreadsheet analysis. Export before making major changes.

All AI features (Strategy Builder, Schedule Analysis, What-If chat) are fully working via a Cloudflare Worker proxy that resolves the browser CORS restriction when calling the Anthropic API directly.

⚡ Features — v1.3

Profile edit + add new profile

Edit name, avatar color, and PIN for the current profile, or add new profiles — all from the nav bar dropdown. No longer requires CardTracker or LoanTracker to exist first.

Calendar modal + ICS export

Monthly calendar showing due days, statement close dates, and projected payoff dates per debt. ICS export for Google Calendar, Apple Calendar, and Outlook. Accessible via button between Open Planner and Add Debt.

Strategy tab + Analysis tab

Strategy Builder moved to its own first tab in the Planner — 5-question questionnaire with Apply This Strategy. What-If chat and Refinance Scenario combined into a single Analysis tab with cross-debt awareness.

Portfolio progress + utilization improvements

Total paid-down progress bar above debt list with 25/50/75/100% milestones. Per-debt progress bars on home screen. Utilization markers at 20% and 80% with color-zoned bars (green/amber/red). Closed account checkbox excludes debts from calculations.

0% APR promo support + morning affirmation

Cards with promotional APR periods show a green badge and use the promo rate in calculations until expiry, then switch to the standard rate. 90-day and 30-day expiry alerts. Morning affirmation splash on every open — context-aware message pools based on payoff progress.

The payoff order list annotates each debt with its lump allocation.

Total Debt card added to portfolio summary

The portfolio summary bar was missing a Total Debt figure. It now appears as the first stat — Total Debt / Monthly Payments / Est. Interest / Payoff Date.

Single Debt tab — current balance stat + CSV export

The Single Debt tab summary row now shows Current Balance as the first stat. The Amortization Schedule panel has a ⬇ CSV export button — columns: Date, Payment, Principal, Interest, Balance.

AI output copy and save buttons

The Schedule Analysis panel now has 📋 Copy and ⬇ Download .txt buttons after the AI response. Each AI message in the What-If chat gets a 📋 Copy button with a 1.5-second "Copied!" confirmation.

🔧 Bug Fixes — v1.1

Standalone profile creation screen

DebtTracker previously assumed a profile already existed from CardTracker or LoanTracker. New users now see a dedicated welcome screen to create their profile — name, avatar color, and Recovery PIN — before entering the app. Existing users are unaffected.

✨ New Module — v1.0

Unified credit card + loan debt tracker

DebtTracker merges CardTracker and LoanTracker into one artifact. Every credit card and installment loan in a single view with one unified payoff schedule.

Cross-debt avalanche and snowball engine

The payoff engine ranks all debts — cards and loans — together. Your extra dollars flow to the highest-priority debt regardless of type. True cross-debt optimization.

Five-tab planner: Schedule, Single Debt, Charts, What-If AI, Progress

Full planner suite covering every angle of debt payoff — strategy, individual amortization, visual charts, AI chat, and payment tracking.

Strategy Builder folded into What-If AI tab

The 5-question AI questionnaire lives inside the What-If tab as a collapsible "Build My Strategy" panel — keeps the tab bar clean without losing the feature.

Debt grouping toggle: by type or interleaved by APR

Default view shows cards first, then loans. Toggle to interleaved mode to see all debts ranked purely by APR — useful for understanding true payoff priority.

One-click import from CardTracker and LoanTracker

A banner on first load detects existing CardTracker or LoanTracker data and offers to import it. No re-entry required — your debts come with you.

Shared profile and API key system

Uses the same cc_profiles and cc_apikey storage as CardTracker and LoanTracker. Set up your profile once — all three modules share it automatically.
