💰 Quick Start Guide · v1.0

Get started with
IncomeTracker

Track every income stream in one place. Know exactly what comes in, how stable it is, and what your real monthly picture looks like.

Getting Started
1

Select Your Profile

IncomeTracker shares your profile from the other FFP modules. If you've already set up a profile in DebtTracker or CardTracker, your name and avatar are already here. Tap the avatar in the top-right to switch profiles.

No profile yet? You'll be prompted to create one on first launch.
2

Add Your Income Streams

Tap + Add Income for each source. Give it a name, pick a type, enter the amount and frequency. That's the minimum — the rest is optional but helpful.

Name → Type → Amount + Frequency → Save ✅
3

Set Stability Ratings

For each stream, mark how reliable it is. This powers the stability indicator in the dashboard and will feed into the AI Advisor when it launches.

✅ Stable 〜 Mostly Stable ⚡ Variable ⚠️ Irregular
4

Review Your Dashboard

The summary at the top shows your total monthly income, annual total, a breakdown by income type, and a stability overview. All values update instantly as you add or edit streams.

One-Time income is shown separately — it's excluded from the monthly recurring total.

💡 Tips for Accuracy
✅

Use after-tax amounts

Toggle "After Tax" on each stream for your take-home pay. This gives you a realistic cash flow picture — especially important once the Spending Module launches.

📅

Set start and end dates

Useful for contract work, seasonal income, or anything with a known end. End-dated streams are still shown but flagged as expiring.

🏷️

Assign a category

Pick from the shared category list. This seeds the platform-wide category system used by Spending and the AI Advisor.

🎨

Color code your streams

Each stream has a color picker. Color-coding by source or type makes the dashboard easier to read at a glance.

📦 Backup & Restore
⬇

Export JSON

Full backup of all income streams. Use this before making major changes or when setting up a new device.

📋

Export CSV

Spreadsheet-friendly export of all streams. Open in Excel or Google Sheets for your own analysis.

📥

Import JSON

Restore from a backup. Choose Replace All (overwrites) or Merge (adds new streams, skips duplicates).
