Getting Started
1

Create Your Profile

Enter your name and choose an avatar color. Set a Recovery PIN — a memorable word like smithfamily. This PIN lets you recover your data on any new device without an account.

Using CardTracker too? Use the same PIN — one profile works across all Financial Freedom Platform modules.

2

Add Your Loans

Tap + Add Loan and choose a type. Enter the current balance, interest rate, and monthly payment — the app auto-calculates remaining months. Add original balance for accurate progress tracking.

🚗 Auto 🏠 Mortgage 🎓 Student 💼 Personal 🏦 Other
Required: Current Balance + Interest Rate + Monthly Payment
3

Add Your API Key

Tap 🔑 in the top bar and paste your Anthropic API key. This unlocks AI Payoff Analysis, Refinance Advisor, and What-If chat for everyone sharing this artifact URL.

Already set in CardTracker? It's the same key — no re-entry needed.
console.anthropic.com → Copy key → Tap 🔑 → Paste
4

Open the Payoff Planner

Tap 📊 Planner in the nav bar (or Open Payoff Planner → from any loan panel). Choose Avalanche or Snowball, add extra monthly budget, and tap Generate Analysis for a personalized AI payoff plan.

Choose method → Set extra budget → AI Plan ✅
Payoff Methods — Which Should You Choose?

🔥 Avalanche — Best for saving money

Attack the highest interest rate loan first. Mathematically optimal — minimizes total interest paid over the life of your loans.

Highest rate first → lowest rate last

❄️ Snowball — Best for motivation

Eliminate the smallest balance first. Each paid-off loan frees up cash for the next. Costs slightly more interest but builds real momentum.

Smallest balance first → largest balance last
Both methods show payoff order, month-by-month table, total interest, and debt-free date side by side in the Planner.
