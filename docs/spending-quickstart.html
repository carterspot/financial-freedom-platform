📊 Quick Start Guide · v1.10

Get started with
SpendingTracker

Import your bank and credit card statements, let AI categorize your spending, and see exactly where your money goes every month.

First Import
1

Export a Statement from Your Bank

Log into your bank or credit card and download a CSV statement for the current month. Most institutions have an "Export" or "Download" option in the transaction history. Chase, BoA, Amex, Wells Fargo — all work.

CSV format required. Any column order is fine — you'll map them in step 3.
2

Tap Import CSV

Click Import CSV in the top-right of the Transactions tab. Create a new account — give it a nickname like Chase Checking or Amex Blue and pick the account type.

The nickname is how SpendingTracker remembers your column mapping for next time.
3

Map Your Columns

The column mapper shows your CSV's column headers. Match them to the required fields: Date, Description, and Amount. Optional: Type field and their category column.

Credit card? Toggle Flip amount sign — CC statements show purchases as positive, but we store expenses as negative. Toggle is remembered for every future import from this account.
4

AI Categorizes Everything

After mapping, AI reviews all uncategorized transactions in one pass. Confident matches are assigned automatically. Anything uncertain goes to a Needs Review queue for you to confirm.

Rules run first → AI handles rest → Review queue → Rules auto-created ✅
The Three Tabs
📊
Summary DEFAULT

Category bar charts

Income categories at top, expenses sorted highest to lowest. Bars show spend as % of monthly income. A dash marks your 3-month rolling average — your visual budget line.

Delta color coding

Green = under average. Amber = 10–25% over. Red = 25%+ over. See at a glance where you're surging this month.

Click to filter

Click any category row to jump to the Transactions tab pre-filtered to that category. No manual setup needed.

📋
Transactions

Monthly transaction list

All transactions for the selected month. Use the month selector arrows to move between months.

Debit / Credit toggle

Filter to All, Debit only, or Credit only. Useful for seeing just expenses or just income credits.

Multi-category filter

Select one or more categories at once. Badge shows how many are active. Works together with the Debit/Credit toggle.

Edit any transaction

Click edit to change category, add notes, lock the category, create a rule from the transaction, or add a new custom category on the spot.

⚙️
Rules

View all rules

See every auto-assignment rule — pattern, category, and whether it was created by you or AI.

Add rules manually

Create a rule: if description contains "WALMART" → Groceries. Contains, starts with, or exact match. Applies to existing transactions immediately.

AI Suggest Rules

Let AI analyze your unruled transactions and propose rules. Review and confirm in one click.

💡 Pro Tips
🔒

Lock categories you've manually set

Click the lock icon on any transaction to prevent AI or rules from ever changing that category. Good for transactions that AI consistently miscategorizes.

📅

Import each account monthly

Re-importing the same account is safe — the deduplication check skips transactions already in the system. You won't get doubles.

⚡

Rules get smarter over time

Every confirmed review item auto-creates a rule. After 2–3 months your imports will be nearly fully automatic — minimal review queue.

🏷️

Use "Create rule from this"

On any transaction row, the edit menu has a "Create rule from this" option. Fastest way to add a rule for a specific merchant.

📦 Backup & Accounts
➕

Multiple accounts

Add as many accounts as you need — checking, savings, credit cards. Each has its own column mapping and sign-flip setting. All transactions appear together in the monthly view.

⬇

Export JSON backup

Export a full JSON backup before making major changes. Includes all transactions, accounts, and rules.

📋

Export CSV

Spreadsheet-friendly export of all transactions. Open in Excel for your own analysis.

Storage note: SpendingTracker is optimized for 1–2 accounts and up to 6–12 months of history in local mode. Cloud sync extends this across devices.
